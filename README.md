# Wind Code

---
## 快速开始
在开始使用前，请先配置您的 Api Key以及Base Url。

您需要将Api Key赋值给 `WindCode_API_KEY`

将Base Url赋值给`WindCode_BASE_URL`
- linux/macOS:
``` bash
export WindCode_API_KEY=<YOUR_API_KEY>
export WindCode_BASE_URL=<YOUR_BASE_URL>
```
- Windows (PowerShell)
``` bash
$env:WindCode_API_KEY="<YOUR_API_KEY>"
$env:WindCode_BASE_URL="<YOUR_BASE_URL>"
```
### 安装以及运行
``` bash
# 克隆仓库
git clone https://github.com/Wind031013/Wind-Code.git
# 进入项目目录
cd Wind-Code
# 安装依赖包
pip install .
# 运行项目
wind
```
