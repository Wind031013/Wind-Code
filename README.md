> **项目描述**：智能 Agent 编程协作工具

## 🚀 快速开始
### 克隆仓库
``` bash
# 克隆仓库
git clone https://github.com/Wind031013/Wind-Code.git
# 进入项目目录
cd Wind-Code
# 安装依赖包
pip install .
# 运行项目
wind
# 单例运行项目
wind 你好
```
在开始使用前，请先配置您的 API 密钥。您需要将密钥赋值给环境变量 `ZHI_PU_API_KEY`。
### linux/macOS:
``` bash
export ZHI_PU_API_KEY=<YOUR_API_KEY>
```
### Windows (PowerShell)
``` bash
$env:ZHI_PU_API_KEY="<YOUR_API_KEY>"
```

## 📚 使用教程
### 指令记忆
可在项目页创建WIND.md文件，将指令写入文件中，WIND会自动读取并执行。

## 版本管理
通过输入`/versions`或`/v`可查看所有带有文件变更的版本。可通过输入`/rollback`或`rb`回滚至指定版本。

## 🛠️ 核心架构(Idea/待实现功能)
> 标注备注： ✅以实现、🔄开发中、⏳ 规划中、❌ 已废弃

### 🔄 智能体循环 (Agentic Loop)
* ✅**思维链规划 (CoT)**：多步任务拆解与逻辑推理。
* **自我修正机制**：基于执行反馈的自动 Debug 与路径调整。
* ✅**Human-in-the-Loop**：关键决策节点的专家干预与确认。

### 🧰 工具系统 (Toolsets)
* ✅**文件系统操作**：支持对文件的 `读`、`写`、`创建`、`删除`、`重命名` 及 `局部更新 (Patching)`。
* ✅**终端交互能力**：具备完整的命令行执行与环境反馈解析能力。

---

### 版本管理
* ✅**版本控制**：在每次进行文件修改时进行版本保存，基于Git进行版本控制，可查看历史版本、回滚至指定版本。

## 🧠 多级上下文管理体系

### 1. 指令记忆 (Instruction Memory)
* ✅**指令约束**：通过读取本地WIND.md文件，读取行为准则，确保输出符合预设规范。

### 2. 短期记忆 (Short-term Memory)
* ⏳**状态机管理**：基于状态机维护当前对话的逻辑上下文，防止任务漂移。
* ⏳**记忆压缩** ：渐进式压缩,每次对话（如果满足条件如：token数量足够）进行会话主题、工具调用情况、工作进度、摘要的提取保存。

### 3. 工作记忆 (Working Memory)
* ⏳**进度跟踪**：实时记录当前任务的阶段性目标与完成情况。
* ⏳**行为纠正库**：
    * *错误追踪*：记录历史报错类型、原因。
    * *经验沉淀*：自动归纳解决方案，避免同类错误再次发生。
* ⏳**项目感知**：深度解析当前技术栈与文件树结构。

### 4. 长期记忆 (Long-term Memory)
* ⏳**用户画像**：记录用户的编程习惯、偏好风格及特定业务领域的背景知识。
